Detailed Description

Abstract parent class for all agents.

Author
Moussa Elgamal, Pascal Imholze, Anton Sprank, Pascal Wittleben

This class is used as parent class for all agents. Child classes gain methods for requesting and offering jobs. Agents are controlled by the control unit.
